Installation Instructions

Follow the prompts below to add Easy deployment into any of your personal or professional projects.

Install CLI

Requirements

  • Windows 10, Mac OSX, Linux
  • Node JS v12 or higher

Install

Mac (Homebrew)

$ brew install easyd-cli

NPM

$ npm install easyd-cli

YARN ADD

$ yarn add easyd-cli
